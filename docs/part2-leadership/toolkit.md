## Governance Checklist

Before deploying any AI system:

- [ ] Use case documented (what it does, what decisions it influences)
- [ ] Risk classification assigned (low / medium / high)
- [ ] Accountable business owner named
- [ ] Data privacy review completed
- [ ] Security review completed
- [ ] Bias / fairness assessment completed (for consequential decisions)
- [ ] Human oversight mechanism defined
- [ ] Monitoring plan in place
- [ ] Rollback / contingency plan defined
- [ ] Regulatory requirements checked
