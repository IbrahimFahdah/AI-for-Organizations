# Chapter 17: Measuring Operational AI Impact

Operational measurement closes the loop between what was promised and what was delivered. Without it, AI programs run on faith rather than evidence.

---

## Adoption Metrics

Adoption metrics measure whether people are using AI effectively:

| Metric | What It Measures | Trap to Avoid |
|---|---|---|
| Active users / eligible users | Adoption breadth | License activation ≠ active use |
| Feature usage depth | How much of the tool is being used | Breadth ≠ value |
| Session frequency | Regular vs. occasional use | Frequency ≠ outcomes |
| Override / correction rate | How often users fix AI output | Zero override = not reviewing |

---

## Productivity KPIs

| KPI | Measurement Method |
|---|---|
| Task completion time | Time-tracked comparison pre/post |
| Documents processed per FTE | Volume metrics over time |
| First-pass quality rate | Rework and revision tracking |
| Cycle time reduction | End-to-end process timing |

All require a pre-deployment baseline. No baseline = no measurement.

---

## Process Efficiency Indicators

Beyond individual productivity, measure process-level outcomes:

- **Throughput** — volume of work completed per period
- **Queue length / backlog** — work waiting to be processed
- **Error rate** — downstream errors attributable to AI outputs
- **Escalation rate** — proportion of AI outputs requiring human review

## Cost Optimization

Track the full cost picture, not just AI spend:

| Cost Component | Pre-AI | Post-AI |
|---|---|---|
| Labor cost per unit of output | | |
| AI API / licensing cost | N/A | |
| QA / review cost | | |
| Error remediation cost | | |
| **Total cost per unit** | | |

If total cost per unit has not improved, the AI deployment has not delivered operational value — regardless of what the AI metrics show.

## Key Takeaways

- Every operational metric needs a pre-deployment baseline.
- Total cost per unit is the operational bottom line — all other metrics lead to it.
- A monthly scorecard with five metrics beats a comprehensive dashboard reviewed quarterly.

**Common mistake:** Measuring AI system performance (accuracy, latency) while ignoring the business process metrics that actually matter.
